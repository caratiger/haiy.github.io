####1.HiveSQL多表join

进行多表联合join是因为建立特征的时候各个特征来源的原始表不同，所以在处理的时候需要将不同表的特征合并
到一起。
通常使用场景是这样的sql：

```
select 
  a.id, b.f1, c.f2, d.f3 
from 
(
    select id from table_1 
) a 
join 
(
  select id, f1 from table_2
) b 
on a.id = b.id
left outer join
(
  select id, f2 from table_3
) c
on a.id = c.id;
```
